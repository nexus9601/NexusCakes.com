## NexusCakes.com

NexusCakes.com is a simple and visually appealing cake showcase website built using HTML. The interface features a clean navigation menu with links to Home, Cakes, Wishlist, Orders, and Contact sections. On the homepage, users can browse a variety of cake flavours including Truffle Cake, Black Forest, Butter Scotch, Red Velvet, Chocolate Cake, Pineapple Cake, Vanilla Cake, and Fruit Cake. The Contact section displays brief details about the brand’s mission — providing customers with authentic taste and flavour — along with a consistent menu and copyright footer.
